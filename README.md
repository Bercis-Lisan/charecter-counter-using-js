# Character Counter (Max 200 Characters)

A simple and interactive Character Counter built using HTML, CSS, and JavaScript. It allows users to type text while displaying the number of characters entered and the remaining characters, with a maximum limit of 200.

## 🚀 Features

- Live character counting
- Maximum limit of 200 characters
- Displays remaining characters
- Prevents users from exceeding the character limit
- Responsive and user-friendly interface
- Clean and lightweight design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 💡 How It Works

1. Enter text into the textarea.
2. The application updates the character count in real time.
3. The remaining character count decreases as you type.
4. Once the limit of **200 characters** is reached, additional input is prevented (or a warning message is displayed, depending on the implementation).
